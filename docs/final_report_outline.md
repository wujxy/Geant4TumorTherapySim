## Ch.2 Theory / 物理机制（新增，~1800 字，5 小节）—— **核心改造**

### 2.1 带电粒子能量损失与 Bragg peak（~400 字）
- **段1（机制）**：Bethe-Bloch
$$-\frac{dE}{dx}=\frac{4\pi}{m_e c^2}\frac{n z^2 e^4}{\beta^2}\left[\ln\frac{2 m_e c^2\beta^2}{I(1-\beta^2)}-\beta^2\right]$$
  物理解释：质子接近射程时 $\beta\to 0$，$1/\beta^2$ 发散（低能截止前）→ $dE/dx$ 急剧上升 → Bragg peak。
- **段2（推论）**：CSDA 射程概念 + 峰后能量耗尽导致急速截止。→ [T2.1] 这预言了 §4.1 质子深度剂量的峰形与峰后近零剂量，是质子宏观剂量学优势的根源。

### 2.2 光子相互作用（~300 字）
- **段1**：光电效应 / Compton / 电子对，能量由次级电子沉积，构成电磁级联。
- **段2（推论）**：注量指数衰减 $I(x)=I_0 e^{-\mu x}$，无"射程"概念，沉积沿路径连续。→ [T2.2] 预言 §4.1 伽马无 Bragg peak、弥散柱状沉积、$f_{\text{tumor}}\approx 5\%\text{–}9\%$ 平台。

### 2.3 $^{10}\mathrm{B}(n,\alpha)^{7}\mathrm{Li}$ 反应与产物射程（~400 字）
- **段1**：反应方程 + 94%/6% 分支比 + Q 值（2.31 MeV 主分支）+ 各产物动能。
- **段2（核心数）**：水中 CSDA 射程（**已复核**）——$\alpha$(1.47 MeV)≈8 μm、$\alpha$(1.78 MeV)≈10 μm；$^{7}\mathrm{Li}$(0.84 MeV)≈4 μm、$^{7}\mathrm{Li}$(1.01 MeV)≈4.5 μm。两产物背对背发射，合计可达 ~12 μm，与细胞直径 10 μm 同量级。→ [T2.3] 这是 BNCT"单细胞杀伤"的物理基础，全文最关键的一个数。

### 2.4 BNCT 双靶选择性与亚细胞几何模型（~400 字）
- **段1**：两层选择性 = 化学靶向（硼载体使肿瘤细胞优先富集 $^{10}\mathrm{B}$）× 物理靶向（α/Li 短射程使高 LET 沉积局限含硼细胞）。
- **段2（几何论证）**：俘获产物近似各向同性发射。uniform：俘获可在核内（$r<2.5\,\mu\mathrm{m}$）直接发生，无穿越损耗；shell：从 $r\in[4,5]\,\mu\mathrm{m}$ 出发，仅朝核半球的立体角有效，且须穿越 ~1.5 μm 胞质 → 预言 shell 核内剂量低于 uniform。→ [T2.4] 预言 §4.2 shell/uniform 核剂量比 ≈ 0.33。

### 2.5 中子输运与方差缩减（~300 字，**诚实反思**）
- **段1**：0.5 eV 中子在水中以氢弹性散射为主要慢化/偏转机制，散射平均自由程 $\lambda=1/\Sigma_t$ 约 0.5–0.7 cm；直射近似下到达概率 $P\sim e^{-\Sigma_t d}$。→ [T2.5] 预言 §4.2 深度扫描到达率随深度指数式下降。
- **段2（方法学反思）**：occurrence bias 通过修改含硼材料内中子 `neutronInelastic` 发生概率 $f$ 倍提高稀有反应采样，对应权重补偿 $w\to w/f$；无偏性的理论条件是反应为稀有独立事件。但在离散微米含硼体（patch 占几何极小部分）中，穿过 patch 的中子极少，偏置后反应由少数高权重径迹贡献，权重分布高度偏斜，且含硼体积边界截面突变造成 weight 不连续 → estimator 依赖 $f$。→ [T2.5] 这解释了 §4.2 浓度扫描未通过倍率独立性验证。

## Ch.5 Discussion（新增，~1800 字，4 小节）—— **核心改造**

- **5.1 两条正交选择性维度（~500 字）**：质子=宏观空间可控（Bragg peak 能量匹配深度）；BNCT=微观细胞选择性（硼富集×短射程）；伽马二者皆弱。论证三者不是同一指标的不同能级，而是正交优化路径。引用 §4.1 + §4.2 证据。
- **5.2 单位俘获响应如何机制性连接细胞剂量（~400 字）**：用 D 实验的 $R_{\text{high-LET}}$（单位俘获核内能量）× 束流俘获数 = 细胞剂量；机制性桥接 D↔B/C，说明为何条件俘获能稳定支撑 BNCT 结论而 biased estimator 不能。
- **5.3 shell 为何弱于 uniform（~400 字）**：不是"shell 更差"，而是反应位点（$r=4\,\mu\mathrm{m}$）到核（$r=2.5\,\mu\mathrm{m}$）的距离 × α/Li 射程（~8/4 μm）共同决定；引申为"$^{10}\mathrm{B}$ 集中分布不是单调更优的设计变量"。
- **5.4 局限性与机制鲁棒性（~500 字）**：列四类局限（非物理浓度、未验证 estimator、零硼正常细胞、纯水 phantom）→ 用施工图 §2 压力测试表论证：因结论是定性机制图像而非定量数值，故这些局限不推翻核心论点。
